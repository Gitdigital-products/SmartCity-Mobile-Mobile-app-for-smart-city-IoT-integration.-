🚀 Overview
SmartCity-Mobile is the interface between urban citizens and the physical world. By integrating IoT sensor data with Solana-based incentives, the app allows residents to interact with city infrastructure, monitor environmental health, and earn rewards for sustainable behavior.
🛠 Features
Live IoT Feeds: Real-time monitoring of air quality, energy grid status, and noise pollution.
Smart Incentives: Earn "EcoTokens" for using public infrastructure or recycling, powered by the FutureProof Toolkit.
Zonal Access: Secure building and area entry using CryptoID verification.
Oracle Mesh: Integration with decentralized oracles to ensure IoT data integrity.
